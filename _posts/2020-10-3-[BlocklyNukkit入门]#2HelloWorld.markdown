## 编写插件

`BlocklyNukkit`以下简称`bn`

bn支持`JavaScript`、`Python`和`Lua`等脚本语言

### JavaScript

```javascript
logger.info("Hello JavaScript!");
```

### Python

```python
print u"Hello Python!"
```

### Lua

```lua
logger:info("Hello Lua!")
```

## 使用插件

放入文件夹`./plugin/BlocklyNukkit`里面
