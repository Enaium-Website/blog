[全民写端]#8简化代码

一. 简化端名和版本

在INSTANCE写

    public String name = "Coreium";
    public String version = "B1";

就可以直接使用Coreium.INSTANCE.name来获取

二. 简化Module

在Module类中加入protected Minecraft mc = Minecraft.getMinecraft(); mc.thePlayer.setSprinting(true);直接使用

三. 简化addSetting

    protected void addSetting(Setting setting) {
        Coreium.INSTANCE.settingManager.addSetting(setting);
    }
